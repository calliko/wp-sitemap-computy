This plugin disables the built-in sitemap.


== Description ==
WordPress 5.5 introduces a new feature that adds basic extensible XML sitemap functionality to WordPress core.
This functionality does not imply settings and all pages are displayed in the sitemap.



== Installation ==

1. Upload dir `wp-sitemap-computy` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
